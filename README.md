# Student Performance Analyzer

Sistema simples em Python para gerenciar notas de alunos pelo terminal.

## Problema
Estudantes precisam acompanhar suas notas para entender seu desempenho escolar.

## Solução
Este projeto permite cadastrar alunos, adicionar notas e calcular médias automaticamente, informando se o aluno foi aprovado ou reprovado.

## Funcionalidades
- Adicionar aluno
- Adicionar notas
- Calcular média
- Mostrar status (Aprovado/Reprovado)
- Salvar dados em JSON

## Tecnologias utilizadas
- Python
- JSON

## Como usar
```bash
python main.py
